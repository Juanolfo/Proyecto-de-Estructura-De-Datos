BioGraph es una plataforma de software desarrollada en Java diseñada específicamente para el modelado y análisis computacional 
de redes de interacción proteína-proteína (PPI) mediante un enfoque de ingeniería de software detallado y artesanal. El sistema 
transforma conjuntos de datos biológicos en grafos interactivos donde cada proteína actúa como un nodo y cada relación funcional 
representa una arista con peso específico, destacándose por utilizar estructuras de datos personalizadas como listas simplemente 
enlazadas y matrices de adyacencia en lugar de colecciones estándar de Java para garantizar un control total sobre la eficiencia 
y el uso de memoria. Gracias a la integración con la librería GraphStream, la herramienta ofrece una visualización dinámica y 
profesional que permite a los usuarios interactuar físicamente con la red, seleccionando o eliminando nodos en tiempo real, 
mientras aplica algoritmos avanzados como Dijkstra para hallar rutas metabólicas críticas y técnicas de recorrido BFS y DFS 
para la identificación de complejos proteicos independientes. Además, el software incluye un análisis de "Hubs" para detectar proteínas críticas 
y un sistema de gestión de persistencia en formato CSV, consolidándose como una solución robusta y educativa para el estudio de 
la arquitectura de redes biológicas y la optimización de algoritmos aplicados a la bioinformática.
